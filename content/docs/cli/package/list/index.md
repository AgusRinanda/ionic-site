# `$ ionic package list`

List your cloud builds
## Synopsis

```bash
$ ionic package list 
```
  
## Details

Ionic Package makes it easy to build a native binary of your app in the cloud.

Full documentation can be found here: **https://docs.ionic.io/services/package/**







## Examples

```bash
$ ionic package list 
```
