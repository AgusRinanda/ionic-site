# `$ ionic ssh use`

{% include fluid/toc.html %}

Set your active Ionic SSH key
## Synopsis

```bash
$ ionic ssh use [<key-path>]
```
  
## Details


Input | Description
----- | ----------
`key-path` | Location of private key file to use
