# `$ ionic ssh add`
Add an SSH public key to Ionic
## Synopsis

```bash
$ ionic ssh add [<pubkey-path>]
```
  
## Details


Input | Description
----- | ----------
`pubkey-path` | Location of public key file to add to Ionic


Option | Description
------ | ----------
`--use` | Use the newly added key as your default SSH key for Ionic
